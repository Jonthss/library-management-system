# 📚 Library Management System

A simple and interactive library management system developed in Java. This command-line interface (CLI) project allows users to register, search, delete, and list books, using the concepts of dynamic lists (`ArrayList`) and in-memory data manipulation.

## 🚀 Features

The system features a navigation menu that offers the following operations:

* **Add New Book:** Allows the registration of the book's title and author, with validation to prevent empty fields.
* **Search Book by Title:** Searches the database and returns the information of the requested book (case-insensitive).
* **Delete Book by Title:** Finds a book and removes it from the database after a user safety confirmation.
* **List All Books:** Displays all registered books and authors in an organized way, or alerts the user if the collection is empty.
* **Exit System:** Safely closes the application.

## 💻 Technologies Used

* **Language:** Java
* **Data Structure:** `ArrayList` (for storing titles and authors)
* **Data Input:** `Scanner` (with exception handling for invalid inputs)
* **Flow Control:** `Enhanced Switch` (Switch Expression) and `do-while` loops.
